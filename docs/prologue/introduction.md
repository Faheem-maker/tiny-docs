## Tiny PHP
Tiny PHP is a simple & minimal framework, modelled to make development of generic looking websites faster and easier. You can generate consistent, usable pages using a built-in library of widgets, enjoy the pre-built authentication & scaffolding features and ship simple websites in hours, not days.

### What is Tiny Best For?
Tiny is meant for small to medium applications that need generic styling, rapid prototyping and a powerful, plug & play feature-set. It's not meant for enterprise grade applications with rigid structure, or highly customized apps that need modified style or use a framework like React.

## Requirements and Prerequisites
Tiny requires PHP 8.0.0 or above and runs best with the latest version of PHP. You can find more detailed requirements for individual features by reviewing the `composer.json` included in each install.

Using Tiny requires basic knowledge of object-oriented programming (OOP), as Tiny is a pure OOP-based framework. It makes use of the latest features of PHP, such as namespaces and traits. Understanding these concepts will help you more easily pick up Tiny.
