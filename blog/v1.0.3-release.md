# What's new in v1.0.3?
In this release, we implement a prototype for migrations, along with several new methods in the database API and file validations.

# Migrations
Migrations allow you to version control your database schema. You can create a migration file using the `make:migration` command.

Example:
```bash
php tiny make:migration create_products_table
```

## Example Migration
Here's an example that creates a migration for a products table:

```php
<?php

namespace app\migrations;

class CreateProductsTable
{
    public function up()
    {
        $tableCmd = db()->createTable('products');
        $tableCmd->id();
        $tableCmd->string('title');
        $tableCmd->string('image');
        $tableCmd->timestamps();
        $tableCmd->execute();
    }

    public function down()
    {
        db()->dropTable('products');
    }
}
```

## Executing Migration
Finally, you can apply the migration using `migrate` command.

```bash
php tiny migrate
```

# File Validations
The release includes 3 new validations:

- **Image**: Verifies that a file is a valid image.
- **Size**: Verifies the size of a file
- **Extension**: Valdiates the extension of a file without verifying its mime type.
