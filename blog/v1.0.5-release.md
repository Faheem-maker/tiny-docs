# What's new in v1.0.5?
The release comes with a bunch of exciting changes. I have renamed the framework, implemented more tests and fixed several bugs.

## Framework Rename
I have renamed the framework to `bolt-php`. This allows us to claim matching github and composer namespaces, and give us a better chance of claiming a domain later on. Since the existing name is already over-used, I'm sure that the updated name will make it easier to access the framework.

## Improved Coverage
We focus on `web` package, improving its coverage to ~**40%**. There are also some tests for blaze for the first time.

I'm attempting to rewrite the blaze package entirely, utilizing proper compilation instead of regex. That's only possible with a solid set of tests.

## Bug Fixes
There are two major bug fixes within this release:

* Fixed a bug with duplicate slash due to leading route slash
* Fixed directory casing for widgets on UNIX

# Next Steps
I'm aiming to communicate the tiny to bolt rename and provide migration steps. I'm also working on beginning the auth rewrite for a pre-built RBAC/ABAC system pre-built to make authentication simpler!

As always, feel free to report any issues you find, try out the new release and let me know your opinions.
